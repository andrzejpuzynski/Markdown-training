## About paragraphs

Wes is **cool**. (bold with asterix)

Wes is __really__ cool. (bold with underscore)

*Wes* is cool. (italic with asterix)

_Wes_ is very cool. (italic with underscore)

Wes is ~~not~~ cool. (strikethrough with tilde)
